# Divar | اپلیکیشن دیوار

## 📝 توضیح کوتاه پروژه

پروژه دیوار است که با الهام از سایت دیوار توسعه داده شده است. کاربران می‌توانند پس از ورود به حساب کاربری خود، آگهی ثبت کنند و آگهی‌های ثبت شده را در صفحه اصلی و پنل کاربری خود مشاهده کنند.

در این پروژه فرآیند احراز هویت و مدیریت دسترسی کاربران به صورت کامل پیاده‌سازی شده است. همچنین یک پنل مدیریت برای ادمین در نظر گرفته شده که امکان اضافه کردن دسته‌بندی‌های آگهی را فراهم می‌کند.

هدف این پروژه، پیاده‌سازی مفاهیم Authentication، Authorization، مدیریت نقش کاربران (Role-Based Access Control) و کار با API در یک سناریوی واقعی بوده است.

## 🛠 روش نصب و اجرای پروژه

### 📥 نصب

1. کلون کردن ریپازیتوری:

```bash
git clone https://github.com/parsamehrpooshan/Divar
```

2. نصب پکیج ها:

```bash
npm install
```

3. ران کردن backend: (بک اند پروژه جداگانه روی گیت‌هاب به آدرس (https://github.com/parsamehrpooshan/divar-api) قرار گرفته است)

4.  اجرای پروژه در حالت توسعه:

```bash
npm run dev
```


📍 سپس پروژه روی آدرس زیر در دسترس خواهد بود:

```text
http://localhost:5173
```
--

کد تایید پس از وارد کردن شماره موبایل در ترمینال بک اند قابل مشاهده خواهد بود.

## 🧪 ویژگی‌های اصلی پروژه

✨ قابلیت‌ها:

* احراز هویت کاربران (Authentication)
* ورود با شماره موبایل و کد تأیید
* ذخیره‌سازی توکن در Cookie
* پیاده‌سازی JWT Authentication
* مدیریت دسترسی کاربران (Authorization)
* محدودسازی صفحات بر اساس Role کاربران
* نمایش آگهی‌های ثبت شده در صفحه اصلی
* ثبت آگهی جدید از طریق داشبورد کاربری
* دسته‌بندی آگهی‌ها
* پنل اختصاصی مدیریت
* امکان ایجاد دسته‌بندی جدید توسط ادمین
* طراحی ساختارمند و ماژولار کامپوننت‌ها

## 👨‍💼 دسترسی ادمین

برای تست بخش مدیریت می‌توانید از شماره زیر استفاده کنید:

```text
09189990099
```

این حساب دارای Role ادمین بوده و به پنل مدیریت دسترسی دارد.
برای رفتن به پنل ادمین به آدرس /admin بروید.

امکانات پنل ادمین:

* مشاهده پنل مدیریتی
* افزودن دسته‌بندی جدید برای آگهی‌ها
